# Dual Trademark Filing Action Sheet

This sheet is for quickly moving the `OPENRELIX` word mark into filing
in both the United States and China. It is not legal advice.

Last reviewed: 2026-04-27.

## Filing Goal

Create the earliest practical application records for the word mark:

```text
OPENRELIX
```

Use standard characters / plain text first. Do not wait for a logo. Do not file
an acronym or nickname alone as the first mark unless trademark counsel clears
it.

File `OPENRELIX` separately in China. A U.S. application does not
automatically protect the mark there. See
`docs/china-chinese-trademark-filing-kit.md`.

## Minimum Same-Day Package

File or hand off these two applications:

- United States: USPTO Trademark Center, International Class 9.
- China: CNIPA China trademark application, International Class 9.

Add International Class 42 only if the brand is also being used for a hosted
online service, SaaS product, or non-downloadable software service. For the
current downloadable CLI / installer project, Class 9 is the clean first filing.

## Owner Data Needed

The applicant must provide or type these values directly into the official form
or attorney / agent intake:

- Applicant legal name.
- Applicant type: individual or legal entity.
- Citizenship, if individual.
- Entity jurisdiction, if company.
- Domicile address.
- Correspondence email and phone, if required.
- Signature name and title.
- Payment method.

Do not store this private owner data in the repository.

## United States Filing Packet

Recommended first application:

- Filing office: USPTO Trademark Center.
- Mark: `OPENRELIX`
- Drawing: standard characters.
- Owner: the individual or legal entity that controls the project brand.
- Class: 9.
- Basis: Section 1(a) use in commerce only if the software is already publicly
  downloadable or installable under the mark; otherwise Section 1(b)
  intent-to-use.

Class 9 goods draft:

```text
Downloadable computer software for creating, organizing, storing, reviewing, and
displaying reusable workflow assets, namely skills, templates, automations, and
task reviews, for use with command-line artificial intelligence developer tools
```

For a Section 1(a) application, prepare a specimen screenshot or webpage print
showing:

- `OpenRelix` used as the source-identifying project name.
- A download, install, release, GitHub repository, npm package, or similar
  action that makes the software available.
- The public URL and access date.

Use Section 1(b) instead of guessing if the first-use dates or specimen are not
ready. Incorrect use claims can create avoidable risk.

## China Filing Packet

Recommended first application:

- Filing office: CNIPA / China trademark system, or a licensed Chinese trademark
  agency.
- Mark: `OPENRELIX`
- Mark type: word mark / plain English characters.
- Owner: the individual or legal entity that controls the project brand.
- Class: 9.
- Goods selection: choose standardized Class 9 software goods in the official
  system where possible, such as downloadable software / recorded computer
  software items under the software subgroup.

Chinese goods draft for agent intake:

```text
第9类：可下载的计算机软件；已录制的计算机程序；计算机软件（已录制）。
用途说明：用于创建、组织、存储、复盘和展示可复用工作流资产，包括技能、模板、自动化和任务复盘，面向命令行人工智能开发工具。
```

Optional Class 42 draft if a hosted service is part of the product plan:

```text
第42类：软件即服务（SaaS）；平台即服务（PaaS）；计算机软件设计；计算机软件咨询。
用途说明：在线提供用于创建、组织、存储、复盘和展示可复用工作流资产的软件服务。
```

If the project later adopts a separate localized brand name, evaluate and file
that name separately.

## Same-Day Execution Checklist

1. Confirm applicant owner.
2. Confirm filing basis for the U.S. application: 1(a) or 1(b).
3. Capture specimen if using U.S. Section 1(a).
4. Do a quick exact-match clearance search for `OPENRELIX`.
5. File the U.S. Class 9 application or send this packet to a U.S. attorney.
6. File the China Class 9 application or send this packet to a Chinese trademark
   agency.
7. Save receipts, serial/application numbers, and filing timestamps outside the
   public repository.

## Agent / Attorney Handoff Note

```text
Please file a word-mark trademark application for OPENRELIX as soon as
possible to establish an application date. The project is an open source,
downloadable CLI / installer system for local-first reusable AI workflow assets.
Please start with Class 9 software goods. Class 42 should be included only if you
recommend it for hosted software-service coverage. Please use standardized goods
descriptions where possible and avoid a logo filing for the first application.
```
