# China Trademark Filing Kit

This is an operational checklist for preparing a fast China trademark filing for
the OpenKeepsake word mark. It is not legal advice.

Last reviewed: 2026-04-28.

## Filing Goal

Create the earliest practical China trademark application record for the mark:

```text
OPENKEEPSAKE
```

File this as a word mark / plain text mark first. Do not wait for a logo. Do not
assume that a U.S. application for `OPENKEEPSAKE` protects the mark in China.

## Short Recommendation

Recommended first application:

- Jurisdiction: Mainland China.
- Filing office: CNIPA / China trademark system, or a Chinese trademark agency
  recorded with CNIPA.
- Mark: `OPENKEEPSAKE`
- Mark type: word mark containing Latin letters.
- Color: no color claim.
- Owner: the individual or legal entity that controls the project brand.
- Primary class: International Class 9.
- Goods count: keep the first filing within 10 goods where possible.

Add Class 42 only if the brand is also used for a hosted online service, SaaS
product, PaaS product, or online non-downloadable software service. For the
current downloadable CLI / installer project, Class 9 is the clean first filing.

Do not add Class 35 as a default. Add it only if the project will operate a
commercial marketplace, advertising service, business-management service, or
other Class 35 activity under the same mark.

## Applicant Route

Do not store owner data in this repository. The applicant or agent should enter
these values directly into the official system or attorney / agency intake.

Use the route that matches the owner:

- Mainland China legal entity: use the entity certificate, such as a business
  license with unified social credit code.
- Mainland China natural person: prepare a valid identity document plus a
  business qualification document, such as an individual-business license, if
  filing an initial trademark registration.
- Foreign individual or foreign entity without habitual residence or business
  office in China: use a legally established Chinese trademark agency.
- Foreign natural person self-filing in China: only consider this route if the
  applicant has the required China residence status and documentation.

If the owner is a U.S. individual or U.S. company, the practical fast path is to
send this packet to a Chinese trademark agency and let the agency file.

## Goods And Services Draft

Use standardized goods / services names from the official China trademark system
where possible. Search the official goods selector under Class 9, especially the
software subgroup, and choose accepted standard names before using custom text.

Class 9 intake draft:

```text
第9类：可下载的计算机软件；已录制的计算机程序；计算机软件（已录制）；可下载的计算机应用软件；计算机程序（可下载软件）。
用途说明：用于创建、组织、存储、复盘和展示可复用 AI 工作流资产，包括技能、模板、自动化和任务复盘；作为可下载命令行软件、安装器和本地资产管理工具提供。
```

Optional Class 42 intake draft only for a hosted service:

```text
第42类：软件即服务（SaaS）；平台即服务（PaaS）；计算机软件设计；计算机软件咨询；计算机软件维护。
用途说明：在线提供用于创建、组织、存储、复盘和展示可复用 AI 工作流资产的软件服务。
```

The official application usually needs selected goods / services, not a long
marketing description. Keep the usage description for agency intake and internal
clarification.

## Form Values

Prepare these values before opening the filing form or sending agency intake:

- Applicant legal name.
- Applicant type: individual or legal entity.
- Applicant nationality or entity jurisdiction.
- Applicant identity / registration document.
- Domicile or registered address.
- Correspondence contact.
- Mark text: `OPENKEEPSAKE`
- Mark type: word mark.
- Color claim: none.
- Mark explanation: `OpenKeepsake` is used as the project source identifier.
- Translation / transliteration: explain only if the form or agent asks.
- Priority claim: none unless counsel confirms a valid priority basis for the
  same mark and goods / services.
- Class: 9 first.
- Optional additional class: 42 only for hosted services.
- Goods / services selected from the official selector.
- Signature / seal information.
- Payment method.

## Clearance Checklist

Do not rely on a general web search. Before filing, search the official trademark
system or ask the agency to search at least:

- `OPENKEEPSAKE`
- `OpenKeepsake`
- Similar English marks containing `Keepsake`
- Similar Chinese transliterations or translations suggested by counsel

Search exact and similar marks in Class 9, and also Class 42 if filing or
planning hosted services. If a conflict appears, keep the filing date goal but
ask the agency for a same-day fallback name or narrower goods strategy.

## Distinctiveness Risk

`OpenKeepsake` is closer to a coined project name than a direct software
description, which is generally stronger than a purely descriptive product
phrase. Counsel should still search for similar `Keepsake` marks and any
localized equivalents before filing.

Risk posture:

- Filing now is still reasonable because the official fee is low and the brand
  is already the intended project name.
- Expect the agent to search for both prior rights and absolute-ground risk.
- Keep fallback brand options available in case the examiner or search results
  show a serious problem.
- Use `OpenKeepsake` consistently in Chinese project docs after filing so evidence
  of source-identifying use can accumulate.

## Same-Day Execution Checklist

1. Confirm whether the owner is a China entity / person or a foreign owner.
2. If foreign-owned, choose a Chinese trademark agency recorded with CNIPA.
3. Run exact and similar searches for the checklist terms above.
4. Select Class 9 goods from the official system, staying within 10 goods if
   possible.
5. File `OPENKEEPSAKE` as a plain word mark with no color claim, or send this packet
   to the agency for filing.
6. Save the application number, receipt, filing timestamp, and agency messages
   outside the public repository.
7. Use `OpenKeepsake™` only where appropriate while pending. Use the registered
   trademark symbol only after registration issues, and only for the registered
   jurisdiction and goods / services.

```text
请优先提交第9类软件商品，尽量使用官方标准商品/服务名称，并尽量控制在10个商品内。只有在您认为托管服务/SaaS覆盖必要时，再建议另报第42类。请先做第9类和第42类的近似检索，并评估“OPENKEEPSAKE”是否存在显著性或近似冲突风险。如风险较高，请当天给出可替代方案；不要等 Logo。
```
