OpenRelix 使用引导

面向 AI coding agents / AI CLI 的本地优先个人资产层。先在 macOS 上用 npm 安装，再让已完成的 agent 工作沉淀为复盘、技能、模板、自动化、受限记忆摘要和私有面板。

快速安装 查看产品展示
terminal
$ npx openrelix install --profile integrated --enable-learning-refresh

需要显式读取 Codex app-server 线程时，在安装命令后追加 --read-codex-app。

快速开始

v0.1 预览版：macOS + Codex CLI 优先支持。

1

确认环境

准备 macOS、Node.js 18+、npm / npx、zsh、Python 3.10+，以及可写的 Codex CLI 用户目录。

2

执行安装

运行上方 npm 命令。安装器会写入用户级配置、安装 repo skill，并按选项注册本地后台任务。

3

查看面板

完成一次整理后，打开本地生成的 overview / panel，核对复盘、资产、记忆摘要和隐私边界。
